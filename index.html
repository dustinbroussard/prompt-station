How this landing page ties the suite together

Each card links to the original app while the workspace iframe keeps them in one place.

  • All app routes are unchanged (`/prompt-enhancer/`, `/llm-open-chain/`, `/prompt-forge/`).
  • Inline view points at those same routes, ensuring service workers, manifests, and caching continue to work.
  • Use the “Open current in tab” link to pop the inline session into its own window.
  • This page is static; no build tooling required—open `index.html` from the repo root.
